IIUSATech AI Bot Intent Editor Help

The Intent Editor is a multiplatform (Windows/OSX/iOS/Android) app intended to help you create and organize intent definitions for conversational bots. This simple help document explains how it works.

Reset

Resets the app by removing all definitions and restoring the app to its default startup condition.

Save

Prompts to save the current intents definitions.

Open

Opens a new definition. Must be a JSON file and properly formatted as an intents definition.

Help

Opens your default browser to this page.

About

Shows the version information of the installed app.

File

Static text showing the name of the file that is currently opened.

Tag

Drop down menu showing the currently selected tag. Selecting the menu shows all available tags from the intents definition.

Edit

Prompts for a new name for the currently selected tag. 

Add

Prompts for the name for a new tag. Creates it with no patterns or responses.

Del

Prompts to confirm if you want to delete the current tag with its current patterns and responses. This action is cannot be reversed.

Patterns

Text editor that allows you to enter one intent pattern per line. 

Responses

Text editor that allows you to enter one response per line.
